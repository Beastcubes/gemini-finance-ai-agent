# gemini-finance-ai-agent
FP&amp;A Variance Agent (ADK + BigQuery + Gemini Enterprise)  An enterprise-grade FP&amp;A analysis agent built using GoogleADK, BigQuery, and Gemini Enterprise.  Provides deterministic, structured financial analysis for Budget vs Actual (BvA) variance reporting, overspend detection, variance contributor ranking, and period-over-period trend analysis.
